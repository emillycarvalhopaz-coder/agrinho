# 🌾 Projeto Agro Forte & Sustentável

Este projeto foi desenvolvido como parte das aulas de programação da **Professora Regiane**. O objetivo da aplicação é criar uma página web moderna e responsiva focada na conscientização sobre o equilíbrio entre a alta produtividade agrícola e a preservação do meio ambiente.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as três tecnologias pilares do desenvolvimento web front-end:

* **HTML5:** Para a estruturação semântica do conteúdo da página.
* **CSS3:** Para a estilização, definição da paleta de cores (tons de verde e terra) e responsividade (Flexbox/Grid).
* **JavaScript (Vanilla):** Para trazer interatividade ao site, incluindo um alerta de boas-vindas e um contador dinâmico que simula métricas sustentáveis em tempo real.

## 📁 Estrutura do Projeto

O código foi organizado seguindo as boas práticas de separação de conceitos:

```text
├── index.html      # Estrutura e conteúdo do site
├── style.css       # Estilização e design responsivo
└── script.js       # Lógica e interatividades em JavaScript

```

## 🛠️ Como Executar o Projeto

1. Faça o download ou clone este repositório.
2. Certifique-se de que os arquivos `index.html`, `style.css` e `script.js` estão todos na **mesma pasta**.
3. Abra o arquivo `index.html` em qualquer navegador web (Chrome, Edge, Firefox, etc.).
